\f0\fs24 \cf0 # Card Guard \uc0\u55357 \u56561 \u55357 \u56594 \
\
Card Guard is an Android mobile application developed as a student project.  \
The app allows users to securely store bank card details and send card freeze requests when a card is lost or compromised.\
\
---\
\
## \uc0\u55357 \u56960  Features\
\
- User authentication using Firebase Authentication\
- Add, view, and delete bank cards\
- Masked card number display for security\
- Freeze card functionality\
- Firestore database integration\
- Email notification draft to user and bank\
- Clean and simple UI using Material Design\
\
---\
\
## \uc0\u55357 \u57056  Technologies Used\
\
- Android (Java)\
- Firebase Authentication\
- Firebase Cloud Firestore\
- Material Design Components\
- RecyclerView\
